# Crop-Yield-Prediction-for-Punjab
A Machine Learning model built to predict the yield of three main crops of Punjab , Wheat, Maize and Rice using the  Meteorological, Soil and Agronomical features


This project is a machine learning based crop yield prediction system developed to estimate agricultural production for different districts in Punjab, Pakistan. The goal of the project is to help improve agricultural planning and decision-making using historical crop and environmental data.

The system predicts crop yield using multiple agricultural and environmental parameters and provides results through an interactive Streamlit web application.

Supported Crops

• Wheat  
• Rice  
• Maize

Target Region

Punjab, Pakistan

Project Objective

The main objective of this project is to use machine learning techniques to estimate crop productivity and analyze how different environmental and agricultural factors affect crop yield.

The system is designed to support:

• Smart farming  
• Agricultural planning  
• Yield estimation  
• Data-driven farming decisions

Technologies Used

• Python  
• Scikit-learn  
• Pandas  
• NumPy  
• Streamlit  
• Matplotlib  
• Seaborn

Machine Learning Models

Different regression models were used and evaluated for crop yield prediction, including:

• Random Forest Regressor  
• XGBoost Regressor  
• Linear Regression  
• Decision Tree Regressor

Features Used

The models were trained using agricultural and environmental parameters such as:

• District  
• Crop area  
• Production values  
• Rainfall  
• Temperature  
• Humidity  
• Seasonal data

Application Features

• User-friendly Streamlit interface  
• Crop yield prediction in real time  
• District-wise estimation  
• Interactive input system  
• Fast prediction results

Dataset

The dataset contains district-level agricultural statistics and crop production data collected for Punjab districts.

Running the Project

Install the required libraries:

pip install -r requirements.txt

Run the Streamlit application:

streamlit run app.py

Deployment

The application can be deployed online using Streamlit Community Cloud.
